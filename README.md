<h1></h1>

**Code Example**

```python
from tprint import *

tprint(tprint.RED, "tprint is dope XD", 7)
```
**Extras**
```
tprint.RED is the colour of the string.

"tprint is dope XD" is the string.

7 is the speed of the text.
```
